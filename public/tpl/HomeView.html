Welcome to Contacts Application (version 2.0.0)

This application works as a test application to demonstrate vREST. We will use vREST to validate REST APIs of this application.

About this application

"Contacts" is a sample CRUD application built with Backbone.js, Twitter Bootstrap, Node.js, Express, and MongoDB. The application allows you to browse through a list of contacts, as well as add, update, and delete contacts. The application has the following schema with some validations: 

            {
                contact: {
                    name: {type: String, required: true, maxLength: 35},
                    email: {type: String, validator: emailValidator},
                    designation: {type: String, maxLength: 35},
                    organization: {type: String, maxLength: 35},
                    country: {type: String, maxLength: 35},
                    aboutMe: {type: String, maxLength: 500},
                    githubId: {type: String},
                    facebookId: {type: String},
                    twitterId: {type: String}
                }
            }

A small note on vREST

vREST is an online application developed by Optimizory to test the REST APIs of a web application under test.

  • A simple, specific and intuitive tool to test your REST APIs.
  • A browser extension (HTTP Recorder), which easily integrates with vREST. And records the HTTP traffic of web application under test and store them as test cases in vREST.
  • An inbuilt runner, which executes the test cases stored in vREST.
  • Beautiful reporting
